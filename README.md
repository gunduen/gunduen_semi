# Gunduen Project
#### Description :taxi: 
> 근두운은 여행 시 좀 더 사용자들이 편안한 여행을 하고 싶은 개발자의 마음을 반영한 프로젝트로 색다른 여행을 제공하고자 하였다.

#### 개발환경
```
- HTML/CSS/Bootstrap
- JDBC
- Apache Tomcat
- Oracle
```
